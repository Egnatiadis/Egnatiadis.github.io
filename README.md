ΠΕΡΙΓΡΑΦΗ ΣΧΕΔΙΑΣΜΟΥ & ΥΛΟΠΟΙΗΣΗΣ:

Το σύστημα JMS (Job Management System) αποτελείται από τρεις βασικές διεργασίες:
τον Συντονιστή (Coordinator), την Κονσόλα (Console) και τις Δεξαμενές (Pools).

1. jms_coord (Συντονιστής):
   Είναι η "καρδιά" του συστήματος. Δημιουργεί τα κεντρικά named pipes (jms_in, 
   jms_out) και περιμένει εντολές από την κονσόλα. Διαχειρίζεται δυναμικά τη 
   δημιουργία Pools. Όταν λάβει εντολή 'submit', ελέγχει αν υπάρχει διαθέσιμο 
   Pool με λιγότερα από 'limit' jobs. Αν όχι, δημιουργεί ένα νέο Pool μέσω 
   fork/exec. Η επικοινωνία με κάθε Pool γίνεται μέσω ξεχωριστών named pipes 
   (π.χ. p0_in, p0_out).

2. jms_console (Κονσόλα):
   Παρέχει τη διεπαφή στον χρήστη. Διαβάζει εντολές από το stdin, τις στέλνει 
   μέσω pipe στον coord και περιμένει την απάντηση για να την τυπώσει.

3. jms_pool (Δεξαμενή):
   Κάθε Pool είναι υπεύθυνο για την εκτέλεση και την παρακολούθηση των εργασιών 
   (jobs). Για κάθε job εκτελεί fork/exec και ανακατευθύνει (dup2) το stdout 
   και stderr σε αρχεία μέσα σε καταλόγους της μορφής:
   outputs_<jid>_<pid>_<date>_<time>


ΣΧΕΔΙΑΣΤΙΚΕΣ ΕΠΙΛΟΓΕΣ:

- Διαχείριση Σημάτων: Έχει υλοποιηθεί handler για το σήμα SIGTERM στα Pools. 
  Όταν ο coord στέλνει SIGTERM στο Pool (κατά το shutdown), το Pool με τη 
  σειρά του στέλνει SIGTERM σε όλες τις ενεργές εργασίες του πριν τερματίσει.
- Κατάσταση Εργασιών: Χρησιμοποιείται η waitpid με WNOHANG και WUNTRACED για 
  να ελέγχεται αν μια εργασία τρέχει, έχει σταματήσει (suspend) ή τελειώσει, 
  χωρίς να μπλοκάρει η διεργασία του Pool.
- Χρονομέτρηση: Αποθηκεύεται το time_t κατά την έναρξη κάθε job για τον 
  υπολογισμό του χρόνου εκτέλεσης στην εντολή status.


ΟΔΗΓΙΕΣ ΜΕΤΑΓΛΩΤΤΙΣΗΣ (COMPILATION):

Για τη μεταγλώττιση των jms_coord, jms_console και jms_pool, αρκεί η εντολή:
    $ make

Για καθαρισμό των εκτελέσιμων και των named pipes:
    $ make clean


ΟΔΗΓΙΕΣ ΕΚΤΕΛΕΣΗΣ:

1. Εκκίνηση Συντονιστή:
    $ ./jms_coord -n <jobs_pool_limit> -l <path_to_outputs>

2. Εκκίνηση Κονσόλας:
    $ ./jms_console -w jms_in -r jms_out

Ενδεικτικές εντολές στην κονσόλα:
    > submit ls -l
    > status <jobid>
    > show-active
    > shutdown


SCRIPT ΣΤΑΤΙΣΤΙΚΩΝ:

Το bash script jms_script.sh δέχεται τις παραμέτρους:
    $ ./jms_script.sh -l <path> -c list
    $ ./jms_script.sh -l <path> -c size [ν]
    $ ./jms_script.sh -l <path> -c purge
